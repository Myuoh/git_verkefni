1. Hvað gera eftirfarandi Linux skipanir?
cd - leyfir þér að færa á milli mappa
ls - sýnir innhald möppu
pwd - sýnir í hvaða möppu þú ert í
mkdir - býr til möppu

2. Hvað gera eftirfarandi Git skipanir?
git clone - "Klónar" git geymslu inn í bashið manns
git status - kíkir hvort það sé ekki örugglega allt rétt í git möppunni
git diff - sýnir hvaða breytingar hafa verið gerðar

3. Hvað gera eftirfarandi Git skipanir saman? Hvaða gagn er að þeirri aðgerð?
git log- sýnir öll commits 
git checkout - leyfir þér að færa þig á milli branch eða endurheimta vinnutréskrár
git branch - sýnir hvaða branch þú ert í

4. Hvað er útgáfustýring (Version Control)?
Útgáfustýring er venjan að rekja og stjórna kóða og breytingum á honum

5. Hverjir eru helstu kostir við að nota GIT?
einfalt að nota, hægt að vita hver gerði breytingar seinast, ókeypis, ef þú gerir einhver mistök geturu tekið allt til baka.

6. Hversu oft telur þú að eigi að gera færslur (commit) í verkefni?
mjög reglulega

7. Hvað er átt við með “Working Directory”?” og “Staging Area” í GIT?
"Working Directory" í git er alltsem á eftir að adda inn  og "staging area" er þar sem öll skjöl fara áður en þú committar þeim
